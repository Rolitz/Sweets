# 🌹 Rolitz Sweets – Landing Page

Website bán hàng online cho thương hiệu **Rolitz Sweets** – chuyên bánh ngọt cao cấp Nhật Bản.

## 🚀 Cách chạy

**Cách 1 – Python (có sẵn):**
```bash
python -m http.server 3000
```
Mở trình duyệt: `http://localhost:3000`

**Cách 2 – Mở file trực tiếp:**
Double-click vào `index.html`

## 📁 Cấu trúc
```
Rolitz/
├── index.html      # Cấu trúc trang
├── style.css       # Toàn bộ CSS
├── script.js       # Cart + animations + form
└── images/         # Hình ảnh sản phẩm
```

## ✨ Tính năng
- Hero section với ảnh bánh tart nổi float animation
- Giỏ hàng (localStorage) với drawer sidebar
- 7 sản phẩm chia 3 nhóm (Bánh / Trầm nụ / Mỹ phẩm)
- Section ưu đãi 23.000 VNĐ
- Brand story section
- Form đặt hàng có validation
- Scroll reveal animations
- Responsive mobile-first
- Toast notification
